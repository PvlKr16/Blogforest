Topic

A topic can be open or closed. An open topic is visible to all users, while a closed topic is visible only to its participants. Users can add members to a topic, including themselves. A user can remove just himself from topic participants. And, it's only the user himself who can remove him from topic participants. Again: anyone who sees the topic can add any registered user as a participant. The participant can remove only himself.

The "unread counter" shows the number of unread topics the user has at the moment. It's renewed automatically every 60 seconds. An unread topic is a topic in which the user is a participant and which they either never opened after it's created, or in which a new comment (or several comments) has been added since their visit. When a user opens such a topic, it becomes read (until a new comment is added). The user can mark a topic as "unread" as a reminder to read it later.

To create a topic, click "+ New" in the header and select Topic.

In the form that opens, fill in the fields (Title and Content are required, Description is optional), and attach files if necessary. If the topic is hidden, uncheck the "Public topic" box. Select members (if necessary) from the list.

Click on Create topic if you haven't changed your mind or Cancel if you have changed your mind.
